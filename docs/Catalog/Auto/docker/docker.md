### 指令详解
+ COPY --> COPY [源路径] [目标路径]
> + 将源路径下的文件/目录下的文件复制到目标路径

+ ADD --> ADD [源路径] [目标路径]
> + 源路径为url链接 --> 下载链接的文件并放入目标路径
> + 源路径为tar压缩文件 --> 自动解压并放入目标路径

+ CMD --> CMD <命令> / CMD ["可执行文件", "参数1", "参数2"...]

+ ENV --> ENV <key>=<value> <key2>=<value2>

+ EXPOSE --> EXPOSE <端口1> (这只是一个申明，并不会实际开启端口服务)
